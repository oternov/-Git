# Подсказка по GIT

### Создание репозитория
```sh
git init
```
#### Добавление файла в GIT
```sh
git add
```
### Сохранение внесенных изменений файла в GIT с сообщением.
```sh
git commit -m "Messsge"
```
### История изменений, внесенных в файл GIT
```sh
* git log (подробно)
* git log --oneline (кратко)
```
* git chekout
* git diff
